# TImagotCXi   CX2

**TImagotCXi** is a small Tamagotchi-style virtual pet game made for the **TI-Nspire CX II** using **Ndless** and **nSDL**.

Take care of your pet by feeding it, playing games, cleaning up after it, giving medicine, turning the lights on/off, and checking its health. The pet grows over time and saves progress between sessions.

## Features

* Virtual pet gameplay for TI-Nspire CX II
* Built with C, Ndless, and nSDL
* Feeding system
* Happiness system
* Simple high/low minigame
* Medicine and sickness
* Cleaning system
* Light on/off option
* Discipline/attention system
* Growth stages
* Simple animated pet sprites
* Lightweight color UI
* Save file support

## Controls

| Button      | Action                    |
| ----------- | ------------------------- |
| Arrow Keys  | Move through menu options |
| Enter       | Select / Confirm          |
| Esc         | Back / Quit               |
| Number Keys | Quick menu shortcuts      |

## Build Requirements

You need:
* TI-Nspire CX II with Ndless installed

Transfer this file to your calculator:

```text
TImagotCXi.prg.tns
```

## Save File

The game creates a small save file:

```text
timagotcxi.sav
```

Delete the save file if you want to fully restart your pet.

## Notes

This project is designed to be lightweight and calculator-friendly. It uses simple sprites, basic colors, and a single main loop to keep performance stable on the CX II.
